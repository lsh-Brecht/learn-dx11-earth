**DirectX 11 Spinning Earth**

This project is a 3D rendering application of a spinning Earth built with DirectX 11 and the native Win32 API. It demonstrates basic graphics programming concepts including 3D transformations, Blinn-Phong shading, texture mapping, and trackball camera controls.

**Note**

Some required header files (such as math and camera/mouse libraries) are not included in this repository. They were originally provided as part of a university assignment and have been excluded to respect copyright and academic integrity policies.

To build the project, you will need to provide your own equivalent libraries.
